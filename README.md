# BanG Dream 圣地巡礼 - 施工中🚧

如果希望拍出第5话的夕阳和夜幕的氛围，请务必提前上官网确认好「[营业时间](https://sunshinecity.jp/zh-tw/aquarium/ticket/calendar.html)」，在九月至三月的冬季里，水族馆只会营业到下午六点。水族馆出去后就不能再进来了，不过顶楼有很多位置可以坐，也有咖啡馆可以填肚子，可以在里面慢慢等夕阳和夜幕降临。

门口有支持中文的自动售票机，但好像只会在人多时开放使用。
